FP0: Final Project Proposal

For my final project, I plan to build off of my game from MP3 to improve the game experience by adding previews for the fruit type before dropping, saving user scores, displaying a leaderboard, and overall mobile and laptop UI improvements.

Sketches

Prototype Version

The prototype version will focus more on the backend of the game, using Firebase to store game data. Along with this, I will create the interfaces for user signup/signin and leaderboard pages. This phase will probably have minimal CSS styling for the new pages.

Final Version

The final version of my project will be where I work on the fruit previews, HTML page adjustments (like adding instructions, improving the "game over" message, etc.) Here I will focus on the frontend design and some of the game experience.

Milestones

  • Implement Firebase--user creation and score data storage
  • Create basic HTML for sign in and score pages
  • Game updates: previewing the fruit before placement, end-line adjustments
  • UI upgrades: fixing "game over" message with button links to replay/view leaderboard, readjustment of page features
  • Mobile compatability
  • Desktop HTML page: add instructions

Unknowns

  • Creating user profile pages? What do users uses to create an account?
  • Increase familiarity with Firebase and user data
  • Tried fried previews during MP3 and couldn't figure it out--need more guidance or time to work on it

Stretch Goals

  • Animation effects
