MP0 Writeup

For this assignment, I was to create a portfolio website, hosted via Github, to document my projects and work throughout the quarter. In a previous course, I already created a portfolio site using Github, so I used what I had from HCDE 439 as a framework to guide me with this time. I started out by creating a new index.html file as a new home page for the site, which could then direct a user to my HCDE 439 or HCDE 438 work. Below is what my site looked like when it started out.

I then used the in-class MP0 example template to help create my HCDE 438 portfolio sub-page. Both of my portfolios required an 'About Me' page, so to consolidate the two, I decided to create a navigation bar in the header which could make the about page accessible on both porfolio sub-pages, as well as function as a shortcut back to the Home page or to my UX portfolio (on a different site).

My last step was working on the CSS to make sure my pages were, in general, consistent with each other, looked okay, and met assignment requirements.

Issues

One issue I had originally was trying to figure out how I would be able to display my projects from both courses on the site; I was choosing between creating a home page and making the two courses as a subpage, or to have the projects divided into two sections on one page instead. I had a bit of trouble figuring out how I should do those, as I had a lot of files already from my 439 portfolio, so I decided to just create one home page that directs the user to the respective course projects. However, because I kept changing the location of the files (in and out of folders, changing names to be more organized, and accessing pages outside of set folders), I struggled a bit with making sure all clicks and links redirected to the correct pages. There were many times while I was developing that I would have to go back and make sure a click was linked to the correct html page.

CSS Transition

For my CSS transition, I decided to work with the title on the home page since it was pretty plain--I wanted to add something interesting to it. I looked up text animations and got a bunch of different types, but I decided to work with color changing as I thought it would fit best with the way my portfolio looked, as something minimalistic but still colorful. In order to make each of the words change to a different color, in the HTML code, each word that I wanted to change had to be separated as a different class, in this case, labeled as "title-word-#". In the CSS, .title-word gave the animation settings, while the style statements for each of the words (1, 2, 3) specified the three colors the words should rotate through.
