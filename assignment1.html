I used four LED lights, in parallel with each other. Each light is supplied with 5V when coded to HIGH. To calculate the resistance needed, I used Ohm's Law (V = I * R). Yellow, green, and red LEDs have an approximate 1.8V drop, so the voltage that actually gets across is 3.2V. And since the current we want is 20 mA, the resister I should use is 160 ohms. I used a larger, standard resister of 220 ohms for each light.

Each LED is connected to a different pin in the circuit to independently control the blinking pattern of the light. Each receives 5V from GND when on HIGH.

// I used Arduino's Blink template code as a starter (hence why some comments are the same) // the setup function runs once when you press reset or power the board void setup() { // initialize digital pin LED_BUILTIN as an output. pinMode(12, OUTPUT); // green1 pinMode(11, OUTPUT); // yellow pinMode(10, OUTPUT); // red pinMode(9, OUTPUT); // green2 } // the loop function runs over and over again forever void loop() { // turns green1 LED on digitalWrite(12, HIGH); // turns yellow LED off digitalWrite(11, LOW); // turns red LED off digitalWrite(10, LOW); // turns green2 LED on digitalWrite(9, HIGH); // wait for half a second delay(500); // turns green1 LED off digitalWrite(12, LOW); // turns yellow LED on digitalWrite(11, HIGH); // wait for one second delay(1000); // turns green1 LED on digitalWrite(12, HIGH); // turns yellow LED off digitalWrite(11, LOW); // turns red LED on digitalWrite(10, HIGH); // turns green2 LED off digitalWrite(9, LOW); // wait for half a second delay(500); // turns green1 LED off digitalWrite(12, LOW); // turns yellow LED off digitalWrite(11, HIGH); // wait for half a second delay(1000); }
